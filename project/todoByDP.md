## 问题记录

1. 为什么service里面request 函数 有的用了 async 而有的没有
2. reducer 里面需要谨记的问题：reducer 会先对比 state 引用地址，都不变默认无 diff.
```js
// 错误示例
addNewUser(state, { payload }) {
    const { data } = state;
    data.push({
        key: new Date().getTime(),
        ...payload,
    });
    return {
    ...state,
    data
    };
},
// 正确示范
addNewUser(state, { payload }) {
    const { data } = state;
    const newData = data.concat({
    key: new Date().getTime(),
    ...payload,
    });
    return {
    ...state,
    data: newData,
    };
},
```
